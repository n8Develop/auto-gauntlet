# Auto-Gauntlet

Auto-play gauntlet game. Three heroes fight endless waves of enemies — fully hands-off, pure spectator. Canvas 2D, rectangle art.

## Setup

```bash
npm install
npm run dev
```

## About

Built as a test project for [Shepherd](https://github.com/n8Develop/shepherd) — exercising phased CLI agent dispatch for multi-phase game development.
